# use-mask-input

## Install

```bash
npm install --save use-mask-input
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'use-mask-input'
import 'use-mask-input/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
