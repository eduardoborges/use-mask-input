# Docusaurus Documentation App

This is the Docusaurus application for the documentation site.

## Structure

- **App code**: Located in `apps/docs/`
- **Documentation markdown files**: Located in `../../docs/docs/` (root `docs/` folder)
- **Blog posts**: Located in `../../docs/blog/` (root `docs/` folder)

## Development

```bash
# From root
npm run dev:docs

# Or from this directory
npm start
```

## Build

```bash
# From root
npm run build

# Or from this directory
npm run build
```
